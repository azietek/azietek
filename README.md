## 🚀 Projects

### 🧬 Oncology Survival Analysis (TCGA-BRCA)
A survival analysis of TCGA breast cancer patients investigating the impact of Triple-Negative Breast Cancer (TNBC) on overall survival.
The project applies Kaplan–Meier estimation, Cox proportional hazards models, and a Weibull AFT model to address assumption violations and improve interpretability of survival effects.
### 🏡 Real Estate Market Analysis
End-to-end data pipeline combining web scraping, predictive modeling, and automation. The system evaluates property pricing fairness using machine learning and delivers personalized weekly recommendations via email.
Focuses on applied ML, automation, and real-world deployment patterns.
### 🎬 Movie Recommendation System
Machine learning-based recommender system that suggests movies based on user similarity and preferences. Built using classical ML techniques and feature engineering.
### 💃 Dance School Database System
Relational database designed for managing dance school operations, including class composition and student levels tracking. Built with a focus on data integrity and usability for non-technical users.
